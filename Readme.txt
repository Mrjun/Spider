如果从github上下载本项目，需要运行以下nuget命令：
Install-Package CefSharp.WinForms -ProjectName Greatzee.Spider.Winform

如果要运行示例目标网站，请运行以下nuget命令
Install-Package Microsoft.AspNet.Mvc -ProjectName SiteDemo
Install-Package Microsoft.AspNet.Mvc.zh-Hans -ProjectName SiteDemo
Install-Package Microsoft.AspNet.Razor -ProjectName SiteDemo
Install-Package Microsoft.AspNet.Razor.zh-Hans -ProjectName SiteDemo
Install-Package Microsoft.AspNet.WebPages -ProjectName SiteDemo
Install-Package Microsoft.AspNet.WebPages.zh-Hans -ProjectName SiteDemo
Install-Package Microsoft.CodeDom.Providers.DotNetCompilerPlatform -ProjectName SiteDemo
Install-Package Microsoft.Web.Infrastructure -ProjectName SiteDemo
